# Aries AI - 项目目录结构与模块分析

## 一、项目根目录

```
phone-AI/
├── app/                          # 主 Android 应用模块
├── core/                         # 公共核心模块
│   ├── common/                   # 通用工具库
│   ├── designsystem/             # 设计系统
│   ├── prompt/                   # Prompt 模板库
│   └── shizuku/                  # Shizuku 桥接库
├── feature/                      # 功能模块
│   ├── settings/                 # 设置功能
│   └── updates/                  # 更新功能
├── accessibility/                # 无障碍服务独立源码
├── Aries-site/                   # 官网与文档站点
├── telemetry-worker/             # Cloudflare Workers 遥测服务
├── xyla/                         # AI 预设配置
├── gradle/                       # Gradle 版本目录
│   └── libs.versions.toml        # 统一版本管理
└── 需求文档/                     # 本项目分析文档
```
