### 3.1 UiAutomationAgent（自动化引擎核心）

```
                    ┌──────────────────────────────────────┐

                    │  负责协调各组件完成 Agent 流程       │
                    │  管理对话历史和上下文                 │

                    │  管理任务状态和进度                   │
                    └──────┬───────────────┬──────────────┘
                           │               │
              ┌────────────┼───────────────┼────────────┐
              │            │               │            │
         ┌────▼────┐ ┌────▼────┐    ┌─────▼─────┐ ┌────▼────┐
         │Action   │ │Action   │    │Screenshot │ │Prompt   │
```

## 五、关键技术设计

### 5.1 流式早停机制
```
模型输出流 → 实时检测是否包含完整动作指令
         ↓ 检测到动作 → 立即停止等待 → 执行动作
         ↓ 未检测到 → 等待完整输出
```

### 5.2 截图优化策略
```
截图请求 → 缓存命中判断 → 节流判断 → 压缩处理 → 返回
         ↓ 未命中     ↓ 通过
         ├ 新截图     ├ 压缩/缩放
         └ 缓存返回   └ 质量控制
```

### 5.3 多级降级兼容
```
虚拟屏模式 (首选) → Shizuku 模式 → 无障碍服务模式 → 纯截图驱动
```

### 5.4 敏感内容保护
```
UI 文本 / 模型输出 / 用户输入
    ↓
敏感关键词检测（支付密码、验证码等）
    ↓
匹配 → 二次确认 / 危险操作拦截
不匹配 → 正常执行
```

## 六、系统服务与权限

| 服务/权限 | 作用 | 依赖 |
|-----------|------|------|
| AccessibilityService | 读取 UI 树、执行手势操作 | 用户手动开启 |
| Shizuku | 虚拟屏创建、高权限系统操作 | Shizuku App + 授权 |
| HiddenApiBypass | 放宽隐藏 API 限制 | Android 9+ |
| 通知权限 | 自动化任务状态推送 | 运行时申请 |
| 悬浮窗权限 | 虚拟屏预览、悬浮聊天 | 运行时申请 |
| Logto OIDC | 用户认证和会员管理 | 可选 |

## 七、遥测系统架构

```
┌──────────────────┐       ┌──────────────────────┐
│  Android App     │──────▶│  Cloudflare Workers  │
│  (心跳上报)      │       │  (遥测服务)           │
└──────────────────┘       └──────────┬───────────┘
                                      │
                                      ▼
                              ┌──────────────────┐
                              │  Cloudflare D1   │
                              │  (SQLite 数据库)  │
                              └──────────────────┘
```

**数据采集**：
- 安装 ID（哈希化）
- 会话 ID
- 平台/版本信息
- 地区（通过 Cloudflare）
- 活跃时间窗口（5 分钟桶聚合）

**统计指标**：
- 当前在线人数（5 分钟窗口）
- 24 小时/7 天活跃用户
- 总安装量
- 版本分布
- 地区分布
- 活跃趋势图
